Question-1: What are the different ways to select element in the DOM?



Answer: We can select element using getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll in the DOM.


Question-2: What is the difference between innerHTML, innerText, and textContent ?



Answer: innerHTML gets or sets the HTML inside an element, innerText gets or sets only the visible text, and textContent gets or sets all text, including hidden ones.


Question-3: What is event delegation in the DOM?



Answer: Event delegation allows us to handle events on a parent element instead of attaching them to multiple child elements. it helps us to make dynamic elements

Question-5: How do you create, add, and remove elements using JavaScript?



Answer: We can create an element using document.createElement, add it using appendChild or insertBefore, and remove it using remove or removeChild.
